# CONGNV TEMPLATE
> **Requires** [Node.js](https://nodejs.org/en/) version 18+. **Requires** [Git](https://git-scm.com/) installed.

## Run
With NPM:

```bash
npm create @congnv0330/template@latest
```

With YARN:

```bash
yarn create @congnv0330/template
```

## Develop

Install package
```
npm install
npm run dev
node index.js
```
